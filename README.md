### Setting Up

* Clone this repo
* Do `npm install` or 'yarn install' to install dependencies

* Make sure you've installed create-react-app locally



### Running The App

Start your app by typing `npm start` or 'yarn start'.
Access the app on <http://localhost:3000>

Run tests with:

```
npm test
```
